UMAP Dimensionality Reduction Mechanism

Uniform Manifold Approximation and Projection (UMAP) is a dimensionality reduction technique that preserves both local and global structures of high-dimensional data using topological data analysis and optimization.

1. Overview of UMAP

UMAP constructs a high-dimensional graph representation of the data and optimizes a low-dimensional graph to match its topological structure. It leverages manifold learning and Riemannian geometry principles.

2. Detailed Steps in UMAP

2.1. Construct High-Dimensional Graph

UMAP models the data as a weighted k-nearest neighbor (k-NN) graph in the high-dimensional space.

  • Step 1: Compute Distances
    Use a distance metric (e.g., Euclidean) to find the k-nearest neighbors for each point \( x_i \).
  • Step 2: Assign Edge Weights
    Calculate similarity weights using a smooth approximation of a Gaussian kernel:
\[ w_{ij} = \exp\left(-\frac{d(x_i, x_j) - \rho_i}{\sigma_i}\right) \]

Where:

  • \( d(x_i, x_j) \): Distance between points \( x_i \) and \( x_j \).
  • \( \rho_i \): Distance to the nearest neighbor of \( x_i \), ensuring local connectivity.
  • \( \sigma_i \): Scaling factor, normalized to control local density.

Step 3: Symmetrize Graph
Symmetrize the weights to create an undirected graph:

\[ w_{ij} = w_{ij} + w_{ji} - w_{ij}w_{ji} \]

2.2. Initialize Low-Dimensional Representation

UMAP initializes a low-dimensional embedding (e.g., 2D or 3D) using spectral embedding or random initialization. A similar k-NN graph is constructed in this space with similarity weights:

\[ v_{ij} = \frac{1}{1 + a \cdot d(y_i, y_j)^{2b}} \]

Where:

  • \( y_i, y_j \): Low-dimensional coordinates of points \( x_i, x_j \).
  • \( a, b \): Hyperparameters controlling the similarity function’s shape (fitted during optimization).

2.3. Optimize Low-Dimensional Embedding

UMAP minimizes a cross-entropy loss to align the low-dimensional graph with the high-dimensional graph:

\[ \text{Loss} = \sum_{i \neq j} \left[ w_{ij} \log\left(\frac{w_{ij}}{v_{ij}}\right) + (1 - w_{ij}) \log\left(\frac{1 - w_{ij}}{1 - v_{ij}}\right) \right] \]

Step 4: Optimization
Use stochastic gradient descent (SGD) to adjust \( y_i \) to minimize the loss, preserving the topological structure.

4. Key Hyperparameters

  • n_neighbors: Balances local vs. global structure (default: 15).
  • min_dist: Minimum distance between points in low-dimensional space (default: 0.1).
  • n_components: Output dimensions (e.g., 2 for visualization).
  • metric: Distance metric for high-dimensional space (e.g., Euclidean).

5. Example: UMAP in Python

import umap
import numpy as np

# Sample high-dimensional data
X = np.random.rand(100, 50)  # 100 points in 50D
reducer = umap.UMAP(n_neighbors=15, min_dist=0.1, n_components=2)
embedding = reducer.fit_transform(X)
print(embedding.shape)  # Output: (100, 2)

6. Advantages of UMAP

Note: UMAP is faster than t-SNE, preserves both local and global structures, and supports various data types.

7. Steps to Apply UMAP

  1. Data Preparation: Preprocess data (e.g., normalize or scale features).
  2. Choose Hyperparameters: Set n_neighbors, min_dist, n_components, and metric.
  3. Construct High-Dimensional Graph: Compute k-NN and assign weights using the similarity function.
  4. Initialize Low-Dimensional Embedding: Use spectral embedding or random initialization.
  5. Optimize Embedding: Minimize cross-entropy loss using SGD.
  6. Output: Use the low-dimensional embedding for visualization or downstream tasks.
